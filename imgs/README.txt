IMAGES FOLDER
=============

This folder should contain the following image files:

1. favicon.png - Favicon for the website (recommended: 32x32 or 64x64 pixels)
2. full-logo.png - Full logo for the navbar (recommended: height 48-60px, transparent background)

Note: If these images are not available, the website will automatically fall back to text-based logos.

Place your logo files here:
- favicon.png
- full-logo.png
